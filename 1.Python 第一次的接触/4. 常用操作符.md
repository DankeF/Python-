# 常用操作符

1. 算术操作符

    ```python
    a = 10
    +  a += 1  //加法     11
    -  a -= 2  //减法     8
    *  a *= 3  //乘法     30
    /  a /= 2  //除       5
    ```
    
2. 比较操作符

    ```python
    ==
    !=
    <
    <=
    >
    >=
    ```
    
3. 逻辑操作符

    ```python
    and &&
    or ||
    not !
    ```

==
**优先级**

> 幂运算（**）
> ↓
> 正负号（+ -） 
> ↓
> 算术操作符（+-*/） 
> ↓
> 比较操作符（< <= > >= == != ） 
> ↓
> 逻辑操作符（not and or）
