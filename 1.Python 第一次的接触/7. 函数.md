# 函数

1. 定义函数

    ```python
    def 函数名():
        函数体
    ```

2. 调用函数

    ```python
    函数名()
    ```

3. 带参数的函数

    ```python
    def 函数名(参数, [参数, 参数, ...]):
        函数体
    ```
    
## 匿名函数

1. 语法

    ```python
    lambda x : 2 * x + 1
    
    g = lambda x : 2 * x + 1
    g(5)
    11
    ```

2. 过滤器

    ```python
    filter(def or None, iterable)
    ```
    
3. map

    ```python
    map(def, iterable)
    ```
